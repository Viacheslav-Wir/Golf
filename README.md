# To play
1. Clone the project.
2. Open the src folder.
3. python3 main.py

# Control
1. "Left", "Right" - change speed
2. "Up", "Down" - change angle
3. "Space" - hit the ball
4. "Backspace" - reset game
5. "Esc" - exit game
